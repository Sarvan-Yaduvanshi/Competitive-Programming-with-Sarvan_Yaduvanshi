## Key Takeaways
- Two Pointer reduces O(N²) to O(N) by using coordinated pointer movement
- **3 main types:** Opposite Direction, Same Direction, Fast & Slow
- Works best when: array is **sorted**, or you need to **compare/pair** elements
- If two pointer doesn't work → try sliding window or binary search
