## Key Takeaways
- Greedy makes the **locally best** choice at each step
- Greedy works when: **Greedy Choice Property** + **Optimal Substructure**
- Two main proof techniques: **Stays Ahead** and **Exchange Argument**
- If greedy doesn't work → usually need DP
