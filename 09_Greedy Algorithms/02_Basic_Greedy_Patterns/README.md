## Key Patterns
- **Sort + Iterate**: Most basic greedy pattern
- **Ratio-based sorting**: Fractional knapsack, weighted scheduling
- **Priority Queue**: Huffman coding, merge patterns
- **Deadline-based**: Job sequencing
