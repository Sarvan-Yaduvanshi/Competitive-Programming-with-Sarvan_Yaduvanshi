# Module 05: Greedy on Arrays

> **Level:** ⭐⭐ Intermediate

| # | File | Topic | Difficulty |

## Key Patterns
- **Farthest Reach**: Track the maximum reachable index (Jump Game)
- **Circular Greedy**: Track cumulative surplus, reset on deficit (Gas Station)
- **Two-Pass Greedy**: Left→Right pass, then Right→Left pass (Candy)
- **Sort + Rearrange**: Rearrange to satisfy constraints (Wiggle Sort)
