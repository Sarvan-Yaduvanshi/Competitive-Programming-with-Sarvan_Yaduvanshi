# Module 06: Greedy on Strings

> **Level:** ⭐⭐⭐ Advanced  
> **Prerequisites:** Strings, Stack, Priority Queue, Module 01-05

## Concepts Covered

| # | File | Topic | Difficulty |
|---|------|-------|-----------|
| 1 | `01_Lexicographic_Greedy_Technique.cpp` | Smallest/largest string construction | Advanced |
| 2 | `02_Remove_Digits_Greedy.cpp` | Monotone stack greedy (Remove K Digits) | Advanced |
| 3 | `03_String_Rearrangement_Greedy.cpp` | Frequency-based PQ rearrangement | Advanced |

## Key Patterns
- **Lexicographic Greedy**: Build smallest/largest string by choosing characters greedily
- **Monotone Stack**: Remove elements to create smallest number (LC 402, 316)
- **Max-Heap Rearrangement**: Place most frequent character first (LC 767, 1405)
- **Queue-based Greedy**: Simulate processes with queues (LC 649)
