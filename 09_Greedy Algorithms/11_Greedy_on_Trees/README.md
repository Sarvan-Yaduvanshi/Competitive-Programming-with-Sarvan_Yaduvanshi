## Key Patterns
- **Leaf-to-Root**: Process leaves first, make decisions bottom-up (Binary Tree Cameras)
- **Root-to-Leaf**: Process root first, propagate greedily down (Sum of Distances)
- **Re-rooting**: Compute answer for one root, then shift to neighbors in O(1)
- **Leaf Pruning**: Iteratively remove leaves (Minimum Height Trees)
