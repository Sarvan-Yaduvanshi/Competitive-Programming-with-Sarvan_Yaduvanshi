## Key Insight
**Matroid Theory** is the mathematical foundation of WHY greedy works.
A matroid is a structure where the greedy algorithm ALWAYS gives the optimal solution.
If your problem has matroid structure → greedy is guaranteed optimal.
If not → you need to verify with exchange arguments or use DP.
