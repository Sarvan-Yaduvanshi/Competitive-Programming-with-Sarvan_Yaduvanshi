## Key Insight
GM-level greedy problems are hard because:
1. **The greedy strategy is not obvious** — requires creative observation
2. **Proof of correctness is non-trivial** — exchange arguments get complex
3. **Often combined with other techniques** — math, number theory, data structures
4. **Construction problems** — you must BUILD a valid solution, not just find a value
5. **Interactive problems** — limited queries, must be strategic
