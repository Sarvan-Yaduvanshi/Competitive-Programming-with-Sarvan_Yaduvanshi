## 🎯 Learning Goals
- Understand what a tree is and why it's important
- Master all tree terminology
- Know every type of tree and when each appears
- Be able to root any unrooted tree
- Read tree input in any contest format

## 💡 GM Tips
- A tree with N nodes ALWAYS has exactly N-1 edges
- Tree = Connected + Acyclic (remember both conditions)
- In CP, trees are almost always given as edge lists — practice reading them fast
