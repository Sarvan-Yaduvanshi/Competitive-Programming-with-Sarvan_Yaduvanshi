# 13 — Advanced DP Patterns

## What You'll Learn
- **Profile DP / Broken Profile DP** — grid tiling, connectivity constraints
- **DP on Permutations** — count valid permutations with constraints
- **Derangements** — permutations with no fixed points
- **Inclusion-Exclusion DP** — count via inclusion-exclusion principle
- **Meet-in-the-Middle DP** — split n into halves, O(2^(n/2))
- **DP with Bitsets** — parallelize boolean DP, O(n×S/64)
- **Rook placement** — counting with forbidden cells
- **Counting Tilings** — domino/tromino tilings using profile DP
