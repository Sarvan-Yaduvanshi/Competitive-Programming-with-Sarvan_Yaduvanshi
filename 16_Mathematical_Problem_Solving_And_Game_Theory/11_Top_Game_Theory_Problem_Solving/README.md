# 📝 Game Theory Cheat Sheet (One-Page Reference)

## 🔥 The Big Three Rules

### Rule 1: Classic Nim
```
k piles: n₁, n₂, ..., nₖ
First wins ⟺ n₁ ⊕ n₂ ⊕ ... ⊕ nₖ ≠ 0
```

### Rule 2: Sprague-Grundy Theorem
```
ANY impartial game = equivalent Nim pile of size G(position)
G(pos) = mex({ G(next) for all valid moves })
For composite games: G(total) = G(g₁) ⊕ G(g₂) ⊕ ... ⊕ G(gₖ)
```

### Rule 3: Backward Induction
```
Terminal (no moves) → LOSING
Can move to any L state → WINNING
All moves lead to W states → LOSING
```

## ⚡ Common Game Patterns

| Game Type | Winner Condition |
|-----------|-----------------|
| Take 1..k from pile of n | L iff n % (k+1) == 0 |
| Classic Nim (multi-pile) | W iff XOR ≠ 0 |
| Staircase Nim | W iff XOR of odd stairs ≠ 0 |
| Wythoff's Game (a, b) | L iff a = ⌊k·φ⌋, b = a + k |
| Moore's k-Nim | L iff all bit sums ≡ 0 (mod k+1) |
| Misère Nim (all piles ≤ 1) | Count parity of piles |
| Misère Nim (some pile > 1) | Same as normal Nim |
| Parity game (only take 1) | Odd total → First wins |
| **Euclid's Game (a, b)** | First to face a ≥ 2b WINS |
| **Chomp (m×n grid)** | First always wins (strategy stealing) |
| **Game on DAG** | G(v) = mex of successors; W iff G ≠ 0 |

## 🔢 Useful Formulas

| Formula | Expression |
|---------|-----------|
| Catalan(n) | C(2n,n) / (n+1) |
| Derangement(n) | (n-1) × (D(n-1) + D(n-2)) |
| Digital Root | 1 + (n-1) % 9 |
| Stars & Bars | C(n+k-1, k-1) |
| Golden Ratio | φ = (1+√5)/2 ≈ 1.618 |
| Fibonacci Matrix | [1,1; 1,0]^n |
| modpow(a, b, m) | O(log b) binary exp |
| modinv(a, p) | a^(p-2) mod p (Fermat) |
| **Inclusion-Exclusion** | \|A∪B\| = \|A\|+\|B\|-\|A∩B\| (generalize with ±) |
| **Burnside Necklaces** | (1/n) Σ_{d\|n} φ(n/d) × k^d |
| **Stirling S(n,k)** | k×S(n-1,k) + S(n-1,k-1) |
| **⌈a/b⌉** | (a + b - 1) / b |
| **Euler Totient** | n × ∏(1 - 1/p) for prime p\|n |

## 🧭 Problem-Solving Decision Tree

```
Is it a two-player game?
├── YES
│   ├── Single pile? → Find pattern mod (k+1) or compute Grundy
│   ├── Multiple independent piles? → XOR Grundy values
│   ├── Both players same moves? → Sprague-Grundy
│   ├── Different moves per player? → Minimax DP
│   ├── Pick from array ends? → Interval DP
│   ├── GCD / Euclidean structure? → Euclid's Game (a ≥ 2b wins)
│   ├── Game on graph/DAG? → BFS backward induction or SG on DAG
│   └── Symmetric structure? → Strategy stealing / mirroring
│
├── Counting problem?
│   ├── Arrange n items → Permutation/Combination
│   ├── Distribute identical → Stars & Bars
│   ├── No one gets own → Derangement
│   ├── Balanced parens → Catalan
│   ├── Count under symmetry → Burnside's lemma
│   ├── Count NOT in any set → Inclusion-Exclusion
│   ├── Partition into subsets → Stirling numbers
│   └── Large nCr → Precompute factorials, mod inverse
│
├── Optimization problem?
│   ├── Maximize product → AM-GM (use 3s)
│   ├── Minimize deviation → Median (|x|) or Mean (x²)
│   ├── Unimodal function → Ternary search
│   ├── Minimize max / maximize min → Binary search on answer
│   └── Constructive ("print any") → Greedy build step by step
│
└── Recurrence / large n?
    ├── Linear recurrence → Matrix exponentiation
    ├── Polynomial values → Lagrange interpolation
    └── Unknown recurrence → Berlekamp-Massey
```
