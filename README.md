### 📖 1. Mastery Guide Files — One File, Complete Mental Model

Each major topic has a `*_Mastery_Guide.cpp` — a single file that walks through **every variant** of a technique:

```
📄 Frequency_Count_Mastery_Guide.cpp covers:
   TYPE 1 → Basic integer frequency count
   TYPE 2 → Character/string frequency
   TYPE 3 → Boolean frequency (tracking seen elements)
   TYPE 4 → Sorted frequency output (using map)
   TYPE 5 → Pair/Tuple frequency count
   ...and 10+ more variants
```

> You don't need 10 different articles. **One file. Complete mental model. Zero confusion.**

### 🔍 2. Pattern-First Thinking

Every solution explains **WHY** the approach works, not just **WHAT** the code does:

```cpp
// 🧠 PATTERN: When you see "find pair with target sum in sorted array"
//    → Think: TWO POINTERS (opposite ends)
//    → Why? Sorted = directional movement possible
//    → Time: O(n), Space: O(1)
```

### 🏗️ 3. Multi-Approach Solutions — Brute → Optimized → Best

```
Each problem file includes:
  ❌ Approach 1 — Brute Force    (understand the problem)
  ⚡ Approach 2 — Optimized      (learn the technique)
  ✅ Approach 3 — Best/Optimal   (master the pattern)
```

## 🚀 Quick Start

### Clone & Go

```bash
git clone https://github.com/Sarvan-Yaduvanshi/Competitive-Programming-with-Sarvan_Yaduvanshi.git
cd Competitive-Programming-with-Sarvan_Yaduvanshi
```

### Build with CMake

```bash
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Debug
make
```

### Compile Any File Directly

```bash
g++ -std=c++17 -O2 -Wall -Wextra -o solution "02_Array & Subarray Techniques/04_Hashing & Frequency Problem Solving/LC_217_Contains_Duplicate.cpp"
./solution
```
